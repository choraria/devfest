# DevFest Finder

**[devfe.st](https://devfe.st)** is a short-link directory for Google Developer Group (GDG) DevFest events worldwide.

- Visit **[devfe.st](https://devfe.st)** to browse all DevFests on a map or in a searchable table
- Use a short link like **[devfe.st/bangalore](https://devfe.st/bangalore)** to go straight to a chapter’s DevFest page

## Request an update

To update a DevFest short link (destination URL, event date, or name), open a GitHub issue:

1. Go to **[Request a DevFest URL update](https://github.com/choraria/devfest/issues/new?assignees=&labels=update-url&template=url_update.yml&title=Update+DevFest+details%3A+%5BCity%5D+%28slug%3A+city-name%29)**
2. Fill in the form — keep the city and slug in the issue title as shown
3. Submit the issue

A maintainer will review and approve the change. Once approved, `devfe.st/your-slug` updates automatically.
